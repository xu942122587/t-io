Vue开发步骤-秒懂

1 安装

1.1   https://nodejs.org/en/download/

2 设置本地库位置

2.1  npm config set cache "D:\nodejs\node_cache"

2.2  npm config set prefix "D:\nodejs\node_global"

3 设置path

3.1  d:\nodejs\node_global;d:\nodejs;

4 用cnpm

4.1  npm install -g cnpm --registry=https://registry.npm.taobao.org

4.2  后面用cnpm代替npm命令就可以了

5 安装vue-cli

5.1  cnpm install -g vue-cli

6 创建项目

6.1  vue init webpack my-project

6.2  cd my-project

6.3  安装依赖

6.3.1   cnpm install

6.4  开始运行

6.4.1   cnpm run dev

6.5  访问项目

6.5.1   localhost:8080

6.6  项目打包

6.6.1   npm run build

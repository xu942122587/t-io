t-io1.7.1发布，让天下没有难开发的即时通讯

t-io简介

  • t-io是基于jdk aio实现的易学、易用、稳定可靠、性能强悍、将多线程运用到极致、内置功能丰富的TCP即时通讯框架(广义上的即时通讯，并非指im)，字母 t 寓意talent。
  • 同类型的框架还有voovan、netty、mina、baseio等，不喜欢t-io的可以去尝试了解这几个，t-io核心源码对所有人按LGPL协议开源，但只对品行良好的开发人员提供技术支持！

社区版更新日志

  1. 增加org.tio.core.intf.PacketListener，packet发送或丢弃后会调用PacketListener.onAfterSent(channelContext, packet, isSentSuccess)
  2. 增加GroupListener，当某条连接路被绑定或解绑到某个Group时触发相关方法
  3. bug fix: 单条链路接收消息数统计未加上
  4. 其它小修小改
